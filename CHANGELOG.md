# Changelog (Slides-Grab Editor Evolution)
작성일: 2026-04-01

## [V1.2.0] - 2026-04-01 (Major Release)

### **핵심 업데이트: "AI 보조에서 사용자 직접 편집 중심으로"**

#### **1. AI BBox 자동 생성 중단 및 선택 모드 최적화**
- **BBox 자동 중단**: 요소 클릭 시마다 발생하던 불필요한 AI Bounding Box 생성을 중단했습니다.
- **수동 선택 우선(Select-First)**: 이제 에디터 실행 시 '선택 모드'가 기본으로 활성화되어, 원하는 요소를 즉시 클릭하고 바로 편집(이동, 수정, 삭제)할 수 있습니다.
- **AI는 요청시에만**: AI는 사용자가 특정 영역을 드래그하여 명확히 편집을 지시할 때만 개입하도록 정책을 변경하여 편집 방해 요소를 제거했습니다.

#### **2. 정밀 드래그 앤 드롭 (Direct Object Move)**
- **자유로운 위치 이동**: 모든 텍스트, 차트, 상자를 마우스로 잡고 원하는 위치로 정밀하게 이동시킬 수 있는 `transform: translate` 기반 드래그 엔진을 탑재했습니다.

#### **3. 동적 컨텐츠 보존형 Undo 시스템**
- **JS Safe Undo**: `Ctrl + Z` 복구 시 차트나 그래프가 사라지는 치명적인 문제를 해결했습니다. 
- **자동 리로드 복구**: 무르기 시 iframe을 지능적으로 새로고침하여 자바스크립트가 그래프를 다시 그리도록 보장합니다.

#### **4. FHD 대형 작업 영역 및 스크롤바 통합**
- **1920x1080 캔버스**: 대시보드 리포트를 1:1 비율로 쾌적하게 볼 수 있도록 FHD 해상도를 기본 지원합니다.
- **이중 스크롤바 제거**: iframe 내부 스크롤을 무력화하고 메인 에디터 스크롤로 통합하여 원본 크기의 결과물을 시원시원하게 탐색할 수 있습니다.

#### **5. 강력한 스타일 주입 (!important Override)**
- **고정 색상 돌파**: 대시보드의 빨간색 성장률 지표처럼 원본에 강하게 고정된 스타일도 `!important` 속성을 활용해 사용자가 원하는 색상으로 강제 변경할 수 있습니다.

#### **6. 실시간 AI 스트리밍 (SSE 기반)**
- **실시간 상태 업데이트**: AI의 작업 상태를 실시간으로 스트리밍하는 SSE(Server-Sent Events) 기술을 도입하여, 편집 요청 시 AI의 진행 상황을 더 안정적이고 빠르게 확인할 수 있습니다.

#### **7. 편의 기능 및 단축키**
- **Esc**: 개체 선택 및 강조 박스 즉시 해제
- **Delete / Backspace**: 선택된 요소 바로 삭제
- **정밀 타겟팅**: 가장 안쪽의 작은 글자까지 정확히 선택 가능하도록 좌표 계산 고도화
